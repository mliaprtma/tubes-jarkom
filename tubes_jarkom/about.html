📡 About Project

Project ini merupakan implementasi sistem Client - Proxy - Web Server menggunakan Socket Programming Python.

🔹 Fitur Sistem

  • HTTP Web Server berbasis TCP
  • UDP Echo Server untuk QoS Testing
  • Proxy Server dengan Cache HIT dan MISS
  • Multithreading Multi-client
  • QoS Analysis: RTT, Packet Loss, Jitter, Throughput
  • Analisis paket menggunakan Wireshark

🔹 Arsitektur Sistem

Client → Proxy → Web Server

Seluruh komunikasi HTTP harus melewati proxy server dan client tidak diperbolehkan mengakses server secara langsung.

🔹 Teknologi yang Digunakan

  • Python Socket Programming
  • TCP dan UDP Protocol
  • Threading
  • HTTP Protocol
  • Wireshark Network Analyzer

🔹 Tujuan Project

Memahami implementasi komunikasi jaringan berbasis socket, analisis Quality of Service (QoS), serta simulasi sistem jaringan Client - Proxy - Server.
